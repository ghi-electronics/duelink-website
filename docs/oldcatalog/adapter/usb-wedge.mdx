A PCB (Printed Circuit Board) USB wedge for connecting DUELink modules to the [USB Interface](../../interface/usb). It includes a power LED and 800mA power regulator.

Note that this is not a standard USB connector, but a printed circuit. This should work on 99% of USB type A connectors with a rare failure.
