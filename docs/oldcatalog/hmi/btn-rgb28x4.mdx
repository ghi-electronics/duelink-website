Function | Description
--- | ---
`SetClr(i,c,l)` | Sets a button at `i` index to `c` color with `l` luminance.
`BtnA()` | Returns 1 if button A is pressed.
`BtnB()` | Returns 1 if button B is pressed.
`BtnC()` | Returns 1 if button C is pressed.
`BtnD()` | Returns 1 if button D is pressed.
