## Prototype

Build prototypes and proof of concepts in minutes vs days, at an extremely low-cost. Let's say you have a quick need for a device that connect to a vehicle with these requirements:
1. Power from a car battery.
1. Capture CAN bus messages.
1. Read current position from GPS.
1. Record a log on a USB memory Stick.
1. Show information on a color display.

Put the soldering iron aside and grab a [Holey Board](./products/axholey) instead! Here is your project done in a couple of hours!

![Vehicle project](pathname:///img/vehicle-project.png)

Each module has a built in library for you! Simple ASCII human-readable commands can show graphics on the display and record files on a USB memory stick! See the [Standard Library](./engine/stdlib) for an impressive list of options.

## A Trusty Connector

By adding a tiny JST connector, you will magically bring a whole level of expandability to your designs. Just imagine how versatile designs will be when you are able to connect any [DUELink Catalog](./catalog/intro) module to your design. 

![downlink](pathname:///img/downlink.png)

This one connector allows you to connect one, or connect several!

![dueduino link](pathname:///img/dueduino-uplink.png)

This is all possible using DUELink [daisylink](./engine/daisylink).

![uplink downlink](pathname:///img/uplink-downlink.png)

You have the option of using either [I2C](./interface/i2c) or [UART](./interface/uart) interface.
