### Script

```batch
_p = 0

while 1
    _c = Dial()

    if _c < _p-1 || _c > _p+1
        for i in Range(_c/3)
        Print("*")
        next
        PrintLn("")
    _p = _c
    end
    
    Wait(10)
wend
```
