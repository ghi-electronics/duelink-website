See [Drivers](/docs/engine/drivers) page for further details.

| Function | Description |
|----------|-------------|
| `SetLed(l, v)` | Sets Led at `l` (1 to 16) to a `v` value (1 on, 0 off). |
| `LedOff()` | Resets all leds to off. |
| `DVer()` | This driver version. |
<details>
<summary><strong>The Code!</strong></summary>

**LED R16 Driver**

```batch
##### LED R16 Driver #####
 
fn DVer()
  return 1.0
fend
 
fn SetLed(l, v) 
    if l > 16
        return
    end
 
    DWrite(l,v)    
fend
 
fn LedOff()
    For l=1 to 17
        SetLed(l,0)
    Next
fend
 
#########################
```

_i = 1

    end
