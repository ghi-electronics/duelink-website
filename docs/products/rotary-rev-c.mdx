### Script

**Script Sample**

```basic
_k = 0 # Previous value
 
while (1)
    _t = GetValue()
     
    if _k != _t
        _k = _t
        PrintLn(_k)
    end
    
    # Read the press using standard lib GPIO 2
    if DRead(2, 1) = 0
        PrintLn("Pressed")
    end
    
wend

```
