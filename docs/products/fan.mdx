<summary><strong>The Code!</strong></summary>

**Fan Driver**

```text
##### Fan Driver #####

fn DVer()
    return 1.0
fend

fn Fan(s)
    if s >100
        s=100
    end
    if s <-100
        s=-100
    end
    
    if s<0
        s=s*-1
        PWrite(3,s/100)
        DWrite(4,0)
    else
        PWrite(4,s/100)
        DWrite(3,0)
    end
fend

#########################
```
