# Makers

---

Whether you are a beginner to circuits or a seasoned expert, DUELink has something to offer.

Beginners can start similar to [Education](education) options. You are a self-learner and we have videos, tutorials and community to help you out.

Seasoned developers can use the system, with its simplicity, or code the on-board micro directly with their specific needs.

The standard unified sizing makes it much easier to design 3D-printed and laser-cut enclosures. See [Specs](../specs) for more details.
