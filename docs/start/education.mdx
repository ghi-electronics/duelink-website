# Education

---

DUELink shines in education. The entire plug-and-play system is user friendly and works for learner of every age.

We recommend the ultra-low-cost [CincoBit](../catalog/mainboard/cincobit) or its OLED display sister [PixoBit](../catalog/mainboard/pixobit). From here you have many [Hosted language](../language/intro) options to command these boards to do whatever you like, or you can simply program the boards directly, with zero software to install, using the [Console](../console).

If [Arduino](../system/arduino) is your choice of IDE, then we recommend [DueDuino](../catalog/mainboard/dueduino).
