# Raspberry Pi

There are multiple ways to take advantage of the DUELink ecosystem with entire Raspberry PI product line. 

## RPI and RPI Zero

We recommend using DuePi Hat. The hat uses [UART](../interfaces/uart) and work with both full-size Raspberry PI and Raspberry PI Zero. It includes a display, a buzzer, terminal block for wiring, and a [Downstream](../interfaces/downstream) socket to extend your RPI with hundreds of sensors and actuators.

Everything is accessible through any of the RPI supported [Coding Options](../coding-options/intro), such as  [Python](../coding-options/python) .

```py
# Show IP address on display
print("Bye DUE!")
# Beep once a second
```

You can also use breakout boards that bring the RPI’s [I2C](../interfaces/i2c) bus to a standard JST socket, like the Sparkfun Qwiic Shim for Raspberry PI (image). https://www.sparkfun.com/sparkfun-qwiic-shim-for-raspberry-pi.html or Sparkfun Qwiic phat https://www.sparkfun.com/sparkfun-qwiic-phat-v2-0-for-raspberry-pi.html 

Tip: When using the Qwiic phat, only use one socket as [Downstream](../interfaces/uart). You can use the other sockets to connect other, non-DUELink things.

## RPI Pico and RP2350/RP2040

Using any of microcontroller-level boards is possible. We recommend using a board with JST connector, like Adafruit QT Py RP2040 https://www.adafruit.com/product/4900 or Sparkfun Pro Micro RP2350 https://www.sparkfun.com/sparkfun-pro-micro-rp2350.html but you can use the [Breakout](/) to wire any Raspberry PI Pico.

:::tip Note to how much power these little circuits can provide vs what DUELink modules are plugged in. Consider Power Inject to add more juice!
:::
