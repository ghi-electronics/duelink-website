## USB Drivers

DUELink modules use standard USB Classes to work. Operating systems should automatically find and load these drivers. However, we are adding them here in case you need them.

Driver | Notes
--- | ---
<a download href="/bin/usb-drivers/win-usb-dfu.zip">Windows USB DFU</a> |  Includes x86 and amd64
<a download href="/bin/usb-drivers/win-usb-com.zip">Windows USB COM</a> |  For virtual serial ports. Includes x8x and amd64
