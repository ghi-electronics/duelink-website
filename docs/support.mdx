## Issue Reporting

Most of the DUELink ecosystem is open-source, through multiple repositories on GitHub. Please post any issues you might find under these repos.

Please do not post questions under GitHub issues, use the [forums](https://forums.ghielectronics.com/c/duelink/31) instead.

Repo | Notes
--- | ---
[DUELink Website](https://github.com/ghi-electronics/duelink-website) | This very website.
[Online Console](https://github.com/ghi-electronics/duelink-console) | The Console, including scripts and firmware update.
[Coding Language Libraries](https://github.com/ghi-electronics/duelink-libraries) | Coding language libraries, from Python and JavaScript to MicroBlocks and Beyond.
[Arduino Libraries](https://github.com/ghi-electronics/duelink-libraries-arduino) | Libraries for controlling DUELink modules right from Arduino-compatible boards.

For firmware related issues, please post on the libraries repo or on the community forum.
