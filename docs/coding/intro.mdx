## Open Source

Libraries for the supported languages are provided open-source and found at the [DUELink Libraries](https://github.com/ghi-electronics/due-libraries) repo. 
We invite you to learn more about the internal workings of [DUELink Engine](../engine/intro.mdx) and also invite you to support additional systems and languages. 

Also, check out the many provided [samples](https://github.com/ghi-electronics/due-samples).


## DUELink Scripts

Beyond using high level programming languages, the DUELink Engine includes [DUELink Script](../engine/scripting.mdx) for special uses.
