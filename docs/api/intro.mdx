# Core API

---

Each DUELink module has its own unique set of commands that are built on top DUELink's [Internal Engine](../engine/intro), with its [Standard Library](../engine/stdlib). These core libraries are provided for multiple languages to give you an easy access to the available core features, through a nicely designed API.

Libraries for the current API are provided for [Python](../language/python.mdx), [JavaScript](../language/javascript.mdx), and [.NET](../language/dotnet.mdx).
