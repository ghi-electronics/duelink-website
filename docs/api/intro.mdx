# API Reference

---


The DUELink core library provided for multiple languages (such as Python) gives the user an easy access to the available feature through a nicely designed API.

| API       | Description          |
| --- |---|
| [Analog](analog.mdx)	| Read or Write analog pins |
| [Bluetooth](bluetooth.mdx)	| Set name or pin code for bluetooth |
| [Button](button.mdx)	| Read a button. Similar to Digital read but handles debounce |
| [Controller Area Network (CAN)](can.mdx) | Read or write message on CAN |
| [Digital](digital.mdx) | Read or write digital pins |
| [Distance](distance.mdx) | Read ultrasonic distance sensor |
| [Frequency](frequency.mdx) | Generate frequency on a specific pin. This uses hardware PWM internally|
| [I2C](i2c.mdx)			| Access the I2C bus for transferring data |
| [Infrared](infrared.mdx) | Read and decode IR remote control signal |
| [LCD](lcd.mdx)			| Draw on LCD (device specific) |
| [LED](led.mdx)			| Control the on-board LED |
| [NeoPixel](neopixel.mdx) | Control smart color LEDs |
| [Pulse](pulse.mdx) | Generate pulse. Great for controlling stepper motors |
| [Script](script.mdx) | Allows for expanding the system with custom scripts for advanced users |
| [Servo](servo.mdx)		| Control servo motors |
| [Sound](sound.mdx)		| Generate sound |
| [SPI](spi.mdx)			| Access the SPI data bus |
| [System Functions](systemfunctions.mdx) | Built-in functions |
| [Temp-Humidity](temp-humidity.mdx) | Works with DHT sensors |
| [Touch](touch.mdx)		| Allows for capacitive touch sensing |
| [UART](uart.mdx)		| Transfer data on the UART serial port |
