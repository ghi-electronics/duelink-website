
# Interfaces

---

Every single DUELink module work with [USB](usb), [I2C](i2c), [UART](uart), and [SPI](spi) on their Upstream socket. Some modules are made in a [microconputer](../catalog/microconputers) form-factor and; therefore, the Upstream socket is replaced with a USB connector.

[Supported Systems](../system/intro) can use the above interfaces with one of the [Hosted Languages](../language/intro).
