# Wireless


![Wireless ESP32 with USB Hook](pathname:///img/wlesp32-usbhook.png)

Cut the wire and use DUELink modules wirelessly! There are several ways to accomplish this, but the easiest is using a wireless connection from a [PC/Laptop](../hw/pc-laptop) or [Phone/Tablet](../hw/phone-tablet) connected to [Wireless ESP32](../products/wlesp32-a).
