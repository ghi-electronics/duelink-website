# Wireless


![Wireless Hook Connection](pathname:///img/wirelesshook-connection.png)

Cut the wire and use DUELink modules wirelessly! There are several ways to accomplish this, but the easiest is using a wireless connection from a [PC/Laptop](../hw/pc-laptop) or [Phone/Tablet](../hw/phone-tablet) connected to [Wireless Hook](../products/adwirelesshook-a).

![Wireless Hook Connection](pathname:///img/wireless-hook.png)

The [Wireless Hook](../products/adwirelesshook-a) module utilizes the virtual serial interface over Bluetooth, just like with [UART Serial](uart) or [USB](usb).
