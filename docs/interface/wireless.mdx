# Wireless

---

Cut the wire and use DUELink modules wirelessly! There are several ways to accomplish this but the easiest one is via using a Bluetooth connection from a [PC/Laptop](../system/pc-laptop) or [Phone/Tablet](../system/phone-tablet) connected to [Bluetooth Hook](../catalog/accessory/bluetooth-hook). 

The Bluetooth Hook module utilizes the virtual serial interface furnished by Bluetooth SSP class and then things work just like they would with [UART Serial](uart) or [USB](usb).
