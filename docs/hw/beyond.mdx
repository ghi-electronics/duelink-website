# Beyond


Have another hardware in mind? Like NVIDIA Jetson Nano, or an ESP32?

![NVIDA Jetson to ESP32](pathname:///img/jetson-esp32.webp)

Any hardware, large or small, should also work with DUELink. As long as that hardware has one of the supported [Interfaces](../interface/intro), then it can take advantage of the the DUELink ecosystem of modules.
