# Beyond


Have another hardware in mind? Like NVIDIA Jetson Nano for example?

![NVIDA Jetson](pathname:///img/jetson.webp)

Any hardware, large or small, should also work with DUELink. As long as that hardware has one of the supported [Interfaces](../interface/intro), then it can take advantage of the the DUELink ecosystem of modules, using any of the supported [Coding Languages](../language/intro).
