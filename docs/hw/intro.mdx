# Supported Hardware

<div style={{ textAlign: 'center'}}>
![Supported Hardware](pathname:///img/duelink-supported-hardware.webp)
</div>

DUELink modules can tether to several types of hardware devices, including [computers](./pc-laptop.mdx), [phones](phone-tablet), [Raspberry Pi](./raspberry-pi), [Arduino](./arduino), [micro:bit](./microbit), and [BeagleBoard](./beagleboard). You have the option to use them with one of the many supported [Coding Languages](../language/intro).

<div style={{ textAlign: 'center'}}>
![Arduino redboard with color display](pathname:///img/redboard-duelink-display.webp)
</div>

Modules tether to these hardware devices over several [Interface](../interface/intro) options, using a common JST connector. Some third-party devices include this connector allowing them to connect directly to any of the DUELink modules.

<div style={{ textAlign: 'center'}}>
![Boards with compatible JST connectors](pathname:///img/supported-hardware-jst.webp)
</div>

We also offer many adapters for popular boards, such as [DuePI](../products/adduepi-b) and [RPILink](../products/adrpilink-a) for [Raspberry Pi](./raspberry-pi).
