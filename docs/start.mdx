## Modules Discovery

It is a good idea to discover information about the connected modules. Start by USB-connecting your module(s) to a PC. If your board does not have a USB connector then use an adapter, like USB Hook.

<div style={{ textAlign: 'center'}}>
![DUELink Connection](pathname:///img/duelink-connection.webp)
</div>

Visit [loader.duelink.com](https://loader.duelink.com/) and use the `Discover` option. Here is an example showing a connected Ghizzy. 

:::tip 
You can click `View Product Details` to go directly to the product's dedicated page.
:::

<div style={{ textAlign: 'center'}}>
![CincoBit](pathname:///img/discover-ghizzy.webp)
</div>

If the "discover" feature failed complaining about a MicroBlocks firmware, then you need to [load the DUELink Official](https://loader.duelink.com/update) firmware.

:::tip
This is a good time to update the firmware if necessary.
:::

## Commanding Modules

DUELink modules understand and execute text commands that are received over [USB](./interface/usb), [I2C](./interface/i2c), or [UART](./interface/uart). To help in processing these commands, we provide helper libraries, with specific "Getting Started", for many [Coding Languages](./language/intro).

## Scripts

All DUELink modules can be loaded with [DUELink Scripts](./engine/scripting). To loading or modify a script, visit [console.duelink.com](https://console.duelink.com/). Click 

<div style={{ textAlign: 'center'}}>
![DUELink Console](pathname:///img/duelink-console.webp)
</div>


Click the `Connect` button. This will access to the first module by default.

<div style={{ textAlign: 'center'}}>
![Connect Button](pathname:///img/console-connect.webp)
</div>

Type `statled(200,200,50)` and click the `Record` button to send the script to the module's memory. This is one of the many available [Standard Library](./engine/stdlib) functions.


<div style={{ textAlign: 'center'}}>
![Record Button](pathname:///img/console-record.webp)
</div>

Click `Play` and the STAT LED will blink 50 times.

<div style={{ textAlign: 'center'}}>
![StatLED](pathname:///img/statled-blink.webp)
</div>

## Arduino

You can load DUELink modules with Arduino programs, directly from the Arduino IDE! This turns the entire DUELink Catalog into Arduino-compatible boards.

You can also connect modules to your Arduino board using I2C, just like you would with Sparkfun Qwiic, Adfruit STEMMA, and Arduino Modulino. The advantage here is that DUELink modules have higher intelligence and execute text commands.


<div style={{ textAlign: 'center'}}>
![Arduino logo](pathname:///img/arduino-logo.webp)
