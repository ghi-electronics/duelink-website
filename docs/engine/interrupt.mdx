
# Interrupts

---

DUELink Engine can run a function when a pin is changed, on interrupts. `IStart(func, pin, trigger)` where trigger is: 0- Falling edge, 1- Rising edge, and 2- Falling & Rising edges.


```py
_i = 450
_b = 0
IStart("Blink", 1, 0)

while 1
  PrintLn("Stat: ", IStat("Blink"))
  Wait(100)
wend

fn Blink()
  DWrite(20, _b&1)# Status LED is always on pin 20
  _b=_b+1
  println("Blink: ", _b)
  if _b=100# stop the interrupt after 100
    IAbort("Blink")
  end
fend
```
