
# Standalone

---


DUELink's [Scripting](./scripting) Language allow modules to run Standalone. The entire [Catalog](../catalog/intro) of modules can run Standalone. This is specifically important on [Mainboards](../catalog/mainboards) such as [John Due](../catalog/mainboard/john-due) and [DueDuino](../catalog/mainboard/dueduino).

image john due flashing eyes GIF!

A Standalone module can also host/command a stream of [Daisylinked](./daisylink) modules.
