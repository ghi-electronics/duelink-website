
# Standalone

---


DUELink's [Scripting](./scripting) Language allow modules to run Standalone. The entire [Catalog](../catalog/intro) of modules can run Standalone. This is specifically important on [microcomputers](../catalog/microcomputers) such as [John Due](../catalog/microcomputer/john-due) and [DueDuino](../catalog/microcomputer/dueduino).

image john due flashing eyes GIF!

A Standalone module can also host/command a stream of [Daisylinked](./daisylink) modules.
