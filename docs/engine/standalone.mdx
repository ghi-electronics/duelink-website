
# Standalone

---


DUELink's [Scripting](./scripting) Language allow modules to run Standalone. The entire [Catalog](../catalog/intro) of modules can run Standalone. This is specifically important on [microconputers](../catalog/microconputers) such as [John Due](../catalog/microconputer/john-due) and [DueDuino](../catalog/microconputer/dueduino).

image john due flashing eyes GIF!

A Standalone module can also host/command a stream of [Daisylinked](./daisylink) modules.
