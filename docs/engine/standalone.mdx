
# Standalone

---

While the main use of DUELink modules is to extend systems with hundreds of actuators and sensors, it is possible that a module can run stand alone. Not only that, that specific module can also become a host to the stream of [Daisylinked](./daisylink) modules.

Take [John Due](../catalog/mainboard/john-due) or any other [Mainboards](../catalog/mainboards) for example. These boards can be used to run Standalone, thanks to 'Recoding Mode' available in the [Internal Engine](../engine/intro).
