# IoT Cloud

![IoT Cloud Services](pathname:///img/iot-cloud.webp)

There are many ways to utilize IoT cloud services. Collect sensor data and send to the "cloud", or use the cloud to control devices remotely.

DUELink modules can be IoT Cloud accessed via one of the [Supported Hardware](/docs/hw/intro) devices, or directly using a [Wireless Interface](../interface/wireless).  

[AWS](./aws) and [Azure](./azure) are great examples.
