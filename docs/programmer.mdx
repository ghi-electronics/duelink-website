The DUELink Programmer is the tool we use internally by our production team for programming and managing DUELink modules. 

The provided to you, along with is [source code](https://github.com/ghi-electronics/duelink-programmer) for reference. We still recommend using the [Console](./console) for firmware update, driver loading, and everything script related!
