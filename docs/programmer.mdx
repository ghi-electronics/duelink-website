# Programmer

The DUELink Programmer is the tool we use internally by our production team for programming and managing DUELink modules. 

It is provided, along with its [source code](https://github.com/ghi-electronics/duelink-programmer) for reference. We recommend using the [Console](./console) for loading scripts and [Loader](./loader) for firmware/driver updates.
