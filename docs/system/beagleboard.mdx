# BeagleBoard
BeagleBoard has many open-source offers and they all work with the DUELink ecosystem.

Image

Any of the [Hosted Language](../language/intro) options can be use on these boards. Depending on the boards, there might be easier way to connect DUELink modules.

## BeaglePlay
This board has a built in JST connector with [I2C](../interface/i2c) signals, I2C5 to be exact! Their [Qwicc Example]( https://docs.beagleboard.org/boards/beagleplay/demos-and-tutorials/using-qwiic.html#beagleplay-qwiic) is a good starting point on how to access the I2C bus. Consider this as a DUELink [Downstream](../interface/downstream) socket.

## BeagleY-AI

This board has a [Raspberry PI](raspberry-pi) compatible footprint/pinout. We recommend using DUELink [DuePI](../catalog/accessory/duepi), which adds a display, a buzzer, a terminal block, and [Downstrem](../interface/downstream) socket.
