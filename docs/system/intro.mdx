# Supported Systems

---

DUELink modules work with several system, from [Raspberry Pi](./raspberry-pi) and [Arduino](./arduino) to [Phones and Tablets](phone-tablet). These systems work with one or more of the supported [Hosted Languages](../language/intro).

:::tip
By saying host, we mean the system commanding a DUELink [Daysylink](../engine/daisylink) of modules.
:::

Modules can also run [Standalone](../engine/standalone) using the internal [Scripting](../engine/scripting) engine.
