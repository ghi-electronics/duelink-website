# DueSTEM Tutorial

DueSTEM's `Input` → `Process` → `Output` mindset shines among [Educators](../educators). You will quickly see how systems take inputs, process that input, to then take action on the outputs. 

While there are several coding options for DueSTEM, we will focus on [MicroBlocks](../language/microblocks) block-coding and on [Python](../language/python). Using other high-level language, like JavaScript and .NET C# is similar to Python. 

Compare | MicroBlocks | Python
--- | --- | ---
Difficulty | Very Easy | Medium ot advanced
Style | Drag-and-Drop blocks | Type coding
PC use | Only to code but runs standalone | PC is needed to execute
Libraries | MicroBlocks libs | DUELink standard libs
Popularity | Mostly School | Global use

## MicroBlocks

Why should you use MicroBlocks:
1. Easy to start
2. Easy to Learn
3. Runs right on the hardware. Programs continue to run even if the computer was disconnected.
4. Any changes on the designer surface show immediately on the device
