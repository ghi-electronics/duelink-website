# L1: First Pixel

## Overview

In this lesson, students will connect [CincoBit](../../products/mccincobit-e) or [PixoBit](../../products/mcpixobit-d) to a computer and begin programming using [MicroBlocks](https://microblocks.fun?utm_source=chatgpt.com).

Students will learn how MicroBlocks communicates with the board in real time and how code changes can instantly update the device without needing to recompile or upload a full program.

Using the display, students will light pixels, create simple images, and explore how computers use small points of light to create graphics and animations.

---

## Learning Goals

Students will:

* connect a board using USB
* connect MicroBlocks to the device
* understand live programming updates
* light LEDs and create simple graphics
* explore coordinates and pixel displays

---

## Time Required

30-45 minutes

## What is MicroBlocks?

MicroBlocks is a live block-based programming environment designed for physical computing devices.

<div style={{ textAlign: 'center'}}>
![microblocks](pathname:///img/microblocks-sample.webp)
</div>

Unlike traditional systems where programs must be compiled and uploaded every time a change is made, MicroBlocks updates the board instantly while the program is running.

This allows students to:

* experiment quickly
* test ideas immediately
* learn through exploration

Changes can happen live while the device is operating.

## Step 1 - Connect the Device

1. Plug CincoBit or PixoBit into the computer using USB.
2. Open MicroBlocks: [microblocks.fun/run/microblocks.html](https://microblocks.fun/run/microblocks.html).
3. Click the connect button.
4. Click on USB.
5. Select the board from the list.

The `Connect` option is on the top right.

<div style={{ textAlign: 'center'}}>
![microblocks-connect](pathname:///img/microblocks-connect-1.webp)
</div>

Select `Connect (USB)` from the menu.

<div style={{ textAlign: 'center'}}>
![microblocks-connect](pathname:///img/microblocks-connect-2.webp)
</div>

Select the `DUELink MicroBlocks` board in the dialog that appears, then click the `Connect` button:

<div style={{ textAlign: 'center'}}>
![microblocks-connect](pathname:///img/microblocks-connect-3.webp)
</div>

If you do not see your `DUELink MicroBlocks` board, make sure it is connected with a good USB cable, then click its reset button.

When the board is connected, the `Connect` button will turn green and the name of the board will appear next to it:

<div style={{ textAlign: 'center'}}>
![microblocks-connect](pathname:///img/microblocks-connect-4.webp)
</div>

MicroBlocks may automatically add some needed compatible libraries, like when CincoBit is connected. 

* the LED Display library
* the Tone library

![boardie libs](pathname:///img/tutorials/cincobit-lessons/boardie-libs.png)

These libraries provide blocks for:

* graphics
* animations
* sounds
* music

**Understanding Live Programming**

MicroBlocks uses:

> live programming

This means:

* programs update immediately
* changes appear instantly
* students can experiment quickly

Unlike many programming tools, there is no compile or download process.

Students can:

* click blocks
* observe results immediately
* modify programs while running

## Step 2 - Showing a Smiley Face

Locate the `display` block inside the `LED display` library. Click on the top side of the block.

> image: keep the block in place and just click on it

When the block is clicked:

* the image immediately appears on CincoBit.

Click the `clear display` block. Boardie's screen will clear out immediately.

Notice:

> The board updates instantly without restarting the program.

This is called **live programming**.

Pull the `display` block into the workspace.

Now:

* edit the pixels
* change the image
* click the block again

The CincoBit display updates instantly.

> image: drag in workspace, modify pixels, click to show on screen.

Students can:

* turn pixels on and off
* draw patterns
* create simple graphics

Create:

## Understanding Pixels

Displays are made from many tiny lights called **pixels**.

* CincoBit uses a 5×5 LED matrix.
* PixoBit uses a larger graphical display with many more pixels.

Programs control pixels by turning them on or off to create:

* images
* text
* animations
* games

## Troubleshooting

* Device does not appear
    * reconnect USB cable
    * try another cable
    * restart MicroBlocks
* Program does not respond
    * verify the board is connected
    * reconnect the device
* Display looks incorrect
    * check pixel positions carefully
    * clear the screen before drawing again
