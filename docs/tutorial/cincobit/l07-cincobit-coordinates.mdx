# L7: Coordinates

## Overview

In this lesson, students will learn how computers use coordinates to position objects on a screen.

Students will explore:

* the X and Y coordinate system
* how displays are organized into rows and columns
* how to place and remove pixels
* how to move objects around the screen

Using buttons, students will control the position of a pixel on CincoBit or move an image on [PixoBit](https://www.duelink.com?utm_source=chatgpt.com).

Students will also combine previous concepts such as:

* variables
* conditions
* events
* loops

to keep objects inside the visible display area.

---

# Learning Goals

Students will:

* understand X and Y coordinates
* identify where (0,0) is located
* use `plot` and `unplot`
* store coordinates in variables
* move objects using buttons
* display coordinate values
* use conditions to keep objects on screen

# Understanding Coordinates

Displays use a coordinate system to identify locations.

Coordinates use:

* X values
* Y values

Example:

```text id="w8m2qf"
(x, y)
```

---

# The X Axis

The X axis controls:

* left and right movement

| Direction | X Value   |
| --------- | --------- |
| Left      | Smaller X |
| Right     | Larger X  |

---

# The Y Axis

The Y axis controls:

* up and down movement

| Direction | Y Value   |
| --------- | --------- |
| Up        | Smaller Y |
| Down      | Larger Y  |

---

# Where is (0,0)?

The coordinate:

```text id="a4k7ty"
(0,0)
```

is located at the:

* top-left corner of the display

This is called:

> the origin

# Plot and Unplot

Students will use:

| Block    | Purpose          |
| -------- | ---------------- |
| `plot`   | Turn on a pixel  |
| `unplot` | Turn off a pixel |

Example:

```text id="v2m8cr"
plot x y
```

lights a pixel at a specific coordinate.

---

# Step 1 — Create Position Variables

Create variables:

```text id="h5q9pb"
x
y
```

Set the starting position:

```text id="k8w1mx"
x = 2
y = 2
```

This starts the pixel near the center.

---

# Step 2 — Draw the Pixel

Use:

```text id="j7n4vd"
plot x y
```

to light the pixel.

---

# Step 3 — Move the Pixel

Use button events to change the X coordinate.

Example:

| Button   | Action     |
| -------- | ---------- |
| Button A | Move left  |
| Button B | Move right |

Example logic:

```text id="z4f2kr"
x = x - 1
```

moves left.

```text id="p9v6tc"
x = x + 1
```

moves right.

Before moving:

* unplot the old position

After moving:

* plot the new position

---

# Display the X Value

Use the `say` block to display:

```text id="r3m8fx"
x
```

Students can observe how the coordinate changes while moving the pixel.

---

# Understanding Screen Limits

What happens if:

* X becomes negative?
* X becomes larger than the screen width?

The object can move off the display.

Programs often use conditions to keep objects visible.

---

# Step 4 — Keep the Pixel On Screen

Use `if` statements to check boundaries.

Example:

```text id="f6x2wj"
if x < 0
```

set:

```text id="d8v1rn"
x = 0
```

Another example:

```text id="m3q7tp"
if x > 4
```

set:

```text id="b9k5cy"
x = 4
```

This keeps the pixel inside the CincoBit display.

# Understanding Movement

Changing coordinates changes object position.

| Change     | Result     |
| ---------- | ---------- |
| Increase X | Move right |
| Decrease X | Move left  |
| Increase Y | Move down  |
| Decrease Y | Move up    |

This is how:

* games
* animations
* robots
* user interfaces

control movement.
