<TabItem value="overview">
This fingertip-sized tiny board is as capable as any other DUELink module. Plug it in a breadboard using male headers, or solder it directly using the edge pads. It includes an upgraded regulator allowing for connecting many DUELink modules on the [Downlink](../../interface/downlink) connector.

This is a great choice to help you in making your own DUELink modules. Don't want USB? Then consider using [Stick](./stick).
