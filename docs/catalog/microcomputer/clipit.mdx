<TabItem value="overview">
[Educators](../../educators) can now elevate their STEM-mission with Clipit. Start immediately with light sensor and touch pads to read inputs. Then use any of the supported [Languages](../../language/intro) to process those inputs and then control the output on the 5x5 LED matrix and buzzer.

Go beyond and add alligator clips for reading touch from objects, or control other circuits.

Of course, this is a DUELink board and [Downlink](../../interface/downlink) is ready for [Daisylinking](../../engine/daisylink) all of your modules.

<TabItem value="drivers">

See [Drivers](../../engine/drivers) page for further details.

Function | Description
---| ---
`TouchA()` | Returns 1 if A pad is touched.
`TouchB()` | Returns 1 if B pad is touched.
`TouchUp()` | Returns 1 if Up pad is touched.
`TouchDown()` | Returns 1 if Down pad is touched.
`TouchLeft()` | Returns 1 if Left pad is touched.
`TouchRight()` | Returns 1 if Right pad is touched.
`Buzzer(f,d)` | Plays buzzer to `f` frequency(50 to 15K) for `d` duration in milliseconds.
`Light()` | Returns voltage level from light sensor.
`DVer()` | This driver version.
