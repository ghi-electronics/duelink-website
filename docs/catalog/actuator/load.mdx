<TabItem value="overview">

The Load module is designed to control up to 8 actuators. No soldering is necessary as all connections are found on a screw terminal block.

Each connection is capable of sourcing up to 80V 1.5A. These can be DC motors, lights, solenoids, relays, pumps and more. The module includes flyback protection diodes to keep your circuits safe when using inductive loads.

All connections are PWM capable allowing for dimming lights and controlling motor speeds.
