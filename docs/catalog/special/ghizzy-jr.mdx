<TabItem value="overview">
Is this the smallest, cutest, and smartest key chain in the world? You decide, but this is a mini robot-shaped pocket-sized microcomputer that plugs right into a USB socket!

It has RGB eyes and a "STAT" mouth as a start, but then the [Downlink](../../interface/downlink) connector will allow Ghizzy Jr to control hundreds of modules!

![Firmware](pathname:///img/catalog/ghizzy-jr-logo.png)

Did you know that Ghizzy Jr is the perfect giveaway for your company? This will not be a giveaway that collects dust in a drawer. Instead, this is an exciting programmable board with endless possibilities. It can even be programmed using the [Arduino IDE](../../hw/arduino).

Note that this is not a standard USB connector, but a printed circuit. This should work on 99% of USB type A connectors with a rare failure.
