<TabItem value="overview">

Ghizzy extends [Ghizzy Jr](./ghizzy-jr) with a buzzer, more LEDs, and a CR2032 battery holder "backpack".

The Uplink [Interface](../../interface/intro) is available via a JST connector or through [USB](../../interface/usb) "feet".

Note that this is not a standard USB connector, but a printed circuit. This should work on 99% of USB type A connectors with a rare failure.

<TabItem value="drivers">

See [Drivers](../../engine/drivers) page for further details.


Function | Description 
--- | ---
`Eye(l,r)` | Sets the `l` left and `r` right "Eyes" with 24bit RGB color (RRGGBB).
`Ear(l,r)` | Activate the `l` left and `r` "ears" LEDs (1 on, 0 off).
`Mouth(o)` | Activate the `m` "mouth" LED (1 on, 0 off).
`BtnLdr()` | Returns 1 if LDR button is pressed.
`Buzzer(f,d)` | Plays buzzer to `f` frequency(50 to 15K) for `d` duration in milliseconds.
`DVer()` | This driver version.
