<TabItem value="overview">
This module breaks down the incoming audio into 63Hz, 160Hz, 400Hz, 1Khz, 2.5Khz, 6.25Khz, 16Khz. It then measures the audio level at these frequencies. You can use these levels as you wish, like using the info to control an LED display.
