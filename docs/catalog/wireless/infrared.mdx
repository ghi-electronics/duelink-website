<TabItem value="overview">

Use light, infrared to be exact, to send and receive data. The data is transferred over a 38Khz frequency to filter out any other surrounding infrared light.

The data sent is compatible with NEC remote control.

Another use of the module is to control NEC compatible devices (like a TV) or receive key presses from a universal remote control.
