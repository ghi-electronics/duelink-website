# Third-party Modules

None has been made yet! Are you making the first one?

---

DUELink is designed with one important goal in mind, to work with all other ecosystems and allow users to make their own modules! We are giving you everything you need to make your own DUELink modules. We only ask you not to use GHI Electronics and DUELink logos without written permission from us, see [Licensing](../licensing) for details.

We have got you covered, from hardware to software

## Hardware
The micro we use on DUELink is STM32C071. You can integrate this micro on your modules and you are half way from having your own DUELink modules! If you are not comfortable using a very tiny QFN32 chip, we have made [DueStick](mainboard/duestick), which brings all the connections to 0.1” headers and castellated holes. It also includes both U and D sockets.
Image

DueStick includes the USB connections on the pads, in case you are deisgning a module that requires a USB connector instead of the U socket. We also offer [DueFlea](mainboard/dueflea) for an ultra-small board with USB C connector.

See the [Specs](../specs) page for more details on the hardware design.

## Software

There isn’t much you need to do on the software front. We have shared the firmware and loader hex files so you can simply load on your own creations, free for commercial and non-commercial use. Everything you need is on the [Downloads](../downloads) page.
