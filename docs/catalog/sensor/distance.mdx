<TabItem value="overview">

This sensor sends an ultrasonic chirp, then it measures the time it takes for the sound to bounce back. The time it takes maps to the distance between the sensor and the object it's facing. This sensor works best on flat solid objects and can measure 40 millimeters to 4 meters.
