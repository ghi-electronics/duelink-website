This gas sensor ships with MQ-7 (carbon monoxide) installed. However, the module uses a socket allowing the sensor be be easily swapped with other gas sensors, like MQ-3 alcohol gas sensor. 

:::warning
Do not use this sensor (or any other ones on this website) for life support applications.
:::
