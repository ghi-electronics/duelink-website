RTC

Realtime Clock

  • Overview
  • Drivers
  • Samples

This module is a RTC (Real Time Clock) "time sensor" with a backup battery to keep time running even when the system is powered off.

Key features

• Real Time Clock
• Button Battery Holder
• 37x22mm overall dimension
