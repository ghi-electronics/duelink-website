The Power Hook works just like the [USB Hook](./usb-hook), but then it also includes a recharging circuit for 3.7V 10440 battery, which is required but not included. Note that the required 10440 battery is the same size as standard AAA batteries, but they are NOT compatible. **NEVER install a AAA battery on this module!**
