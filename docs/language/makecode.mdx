# MakeCode

---

image

For users who enjoy block-coding, MakeCode allows [micro:bit](../system/microbit) users to utilize hundreds of DUELink modules.

To connect DUELink modules, you need a micro:bit accessory that adds a JST connector, like the one we use for [Downstream](../interface/downstream). For example, [Sparkfun Qwiic micro:bit Breakout](https://www.sparkfun.com/sparkfun-qwiic-micro-bit-breakout.html) is a good option.

image

This board connects the JST connector to [I2C](../interface/i2c) bus, which is one of supported [Interfaces](../interface/intro).

Lets get started! Create a MakeCode project like you always did on [makecode.microbit.org](https://makecode.microbit.org/).

image


This example blinks the status LED 50 times on any downstream module

```js
function SendCommand (str: string) {
    pins.i2cWriteBuffer(0x1A, Buffer.fromUTF8(str), false)
}
SendCommand("LED(200,200,50)")
```
