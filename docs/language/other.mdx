# Other Languages

---

![And More](/img/other-languages.png)

Any system with serial port access can utilize a DUELink modules. If [Excel](excel) can work with DUELink, anything else can!!

There are plenty of codes and samples to help you understand and port to whatever you like! See [DUELink Samples](https://github.com/ghi-electronics/duelink-samples) and [DUELink Libraries](https://github.com/ghi-electronics/duelink-libraries) repos.
