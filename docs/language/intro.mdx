 # Hosted languages

---
<div class="img-center">![languages](../images/languages.png)</div>

A long list of coding language options, including [Python](./python) and [JavaScript](./javascript), can be used on several [Supported Systems](../system/intro).

We call these "Hosted" because they do nto run on modules directly. They run on one of the [Supported Systems](../system/intro) with provided support allowing these systems to take advantage of the entire [Catalog](../catalog/intro).

:::tip
Beyond hosted languages, DUElink modules can run [Standalone](../engine/standalone) using powerful [Internal Engine](../engine/intro) with a tiny [Scripting](../engine/scripting) language.
:::
