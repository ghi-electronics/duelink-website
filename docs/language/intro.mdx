# Languages


![languages](pathname:///img/languages.png)

Several coding languages work with DUELink, including [Python](./python), [MicroPython](./micropython), [JavaScript](./javascript), and [.NET](./dotnet.mdx). Block coding is also possible using [MicroBlocks](./microblocks), [MakeCode](./makecode), and [App Inventor](./app-inventor).


DUELink modules work with a [supported hardware](../hw/intro).

![Daisylink](pathname:///img/daisylink-normal.png)

Each DUELink module is also a self-contained mini-computer that can run [standalone](./standalone).

![Daisylink with Power](pathname:///img/host-mode.png)
