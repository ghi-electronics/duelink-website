# Languages



Several coding languages work with DUELink, including [Python](./python), [MicroPython](./micropython), [JavaScript](./javascript), and [.NET](./dotnet.mdx). 

![languages](pathname:///img/languages.webp)


Block coding is also possible using [MicroBlocks](./microblocks), [MakeCode](./makecode), and [App Inventor](./app-inventor).

![MicroBlocks](pathname:///img/microblocks-sample.webp)


DUELink modules work with a [supported hardware](../hw/intro).

![Supported Hardware](pathname:///img/duelink-supported-hardware.webp)


Each DUELink module is also a self-contained mini-computer that can run [standalone](./standalone). Modules can also be daisylinked togther.

![daisylink](pathname:///img/daisylink-normal.webp)
