# Languages

DUELink modules are self-contained minicomputers, allowing modules to work in two different ways: You can program modules directly to run [Standalone](./standalone). Or you can Tether module(s) to a [Supported Hardware](../hw/intro) and code it one of the several coding languages, including [Python](./python), [MicroPython](./micropython), [JavaScript](./javascript), and [.NET](./dotnet.mdx).
