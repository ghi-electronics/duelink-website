 # Hosted languages

---
![languages](/img/languages.png)

Several coding languages work with DUELink, including [Python](./python) and [JavaScript](./javascript). We call these "Hosted" because they do not run on modules directly. They run on one of the [Supported Systems](../system/intro).

Beyond hosted languages, DUElink modules can run [Standalone](../engine/standalone) using:
1. The modules' built-in [Internal Engine](../engine/intro) with a tiny [Scripting](../engine/scripting) language.
2. [Arduino](../system/arduino)
