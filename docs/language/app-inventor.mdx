# App Inventor

---

[App Inventor](https://appinventor.mit.edu/) allows anyone, including children, to building mobile apps with ease.

There are 2 ways to access DUELink modules from App Inventor, USB and Bluetooth.

## USB

This only works with Android as iOS does not support the standard USB CDC Class. As for the modules, you can either use one of the modules with built in USB, such as the ultra-low-cost [CincoBit](../catalog/mainboard/cincobit) or add [USB Hook](../catalog/accessory/usb-hook).

## Bluetooth

Bluetooth is a bit easier as there is no physical connection to the phone or tablet. These devices can pair with [Bluetooth Hook](../catalog/accessory/bluetooth-hook), which in turn brings the stream of [DaisyLink](../engine/daisylink) devices to the [Wireless](../interface/wireless) connection.
