##### CincoBit Driver #####

1. Press button A, all leds should be off, a beep will play as long as button A is pressed
2. Press button B, all leds should be on, a beep will play as long as button B is pressed
3. Cover light sensor, all leds should be off
4. Use flash light points directly to light sensor, all leds should be on
5. Row of leds also tell light status:
   One row is on: ~10-20%. Two rows are on: 20-40%. Three rows are on: 40-60%
   Four rows are on: 60-80%. All rows are on: 80-100%
   
   In a normal room, light is about 15-30%, so first row is on, and the second row may blinking
