##### Snowy #####

1.Make sure the status LED is blinking.
2. Make sure all piano keys work by touching any pad. A tone will play.
3. Touch the left arrow: all LEDs (1, 2, 3, 4, and 5) will turn off, and short beeps will play.
4. Touch the right arrow: all LEDs (1, 2, 3, 4, and 5) will turn on, and short beeps will play.
5. Press the LDR button. Short beeps will play.
