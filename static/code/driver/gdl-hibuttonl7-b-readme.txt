##### Button L7 Test #####

1. Make sure status LED is blinking first.
2. Press button Up, the 'up' led will be on, as long as button Up is pressed and hold down.
3. Repeat step (2) for button Down, Left, Right, Back and Forward buttons.
4. Press Enter (or center button), all button leds will be on. Release this button, all leds will be off.
