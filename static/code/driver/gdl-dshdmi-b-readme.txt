##### HDMI readme #####

1. Make sure the status LED is blinking.
2. Make sure the monitor displays the pattern of colors: White, Yellow, Cyan, Green, Magenta, Red, Blue
3. Make sure the text DUELink appears in the center of the monitor.
4. Make sure everything displayed on the monitor is clear, with no visual noise.
5. A beep will be played three times after the picture appears on the monitor.
