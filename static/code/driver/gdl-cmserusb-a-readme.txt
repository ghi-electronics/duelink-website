##### SerialUSB #####

1.Make sure status LED is blinking first.
2.Open Tera Term and connect to the new port using the default configuration: 9600 baud rate, 8 data bits, 1 stop bit, no parity.
3.Whatever tester type in Tera Term, the device will respond with that character plus one (type 'A', respond 'B', type 'a', respond 'b'))
