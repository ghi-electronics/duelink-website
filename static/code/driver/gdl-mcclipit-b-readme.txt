##### CincoBit Driver #####


1. Touch and release on pads: A, B, Up, Down, Left or Right, the test kit screen show what pad has been touched.
2. Every touched on pad detected, buzzer will play beep, and all leds will be on.
3. Cover the light sensor, all leds will be off.
4. Use flash light points directly to light sensor, all leds should be on.
