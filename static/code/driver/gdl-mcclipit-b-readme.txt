##### CincoBit Driver #####


1. Touch and release on pads: A, B, Up, Down, Left or Right, the test kit screen shows what pad has been touched.
2. Every touched on pad detected, buzzer will play beep, and all leds will be on.
3. Press button LDR, the test kit screen shows buzzer LDR is pressed, buzzer will play beep, and all leds will be on.
4. Cover the light sensor, all leds will be off.
5. Use flash light points directly to light sensor, all leds should be on.
6. Row of leds also tell light status:
   One row is on: ~10-20%. Two rows are on: 20-40%. Three rows are on: 40-60%
   Four rows are on: 60-80%. All rows are on: 80-100%.
   
   In a normal room, light is about 15-30%, so first row is on and the second row may blinking
