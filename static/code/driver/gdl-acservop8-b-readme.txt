##### Servo P8 readme #####

1.Make sure the status LED is blinking.
2.Connect the servo motor to a single pin from P1 to P8 — one pin at a time.
3.Each pin will drive the motor to 0°, 45°, 90°, 135°, and 180°.
4.Then the pin will drive the motor back from 180° to 135°, 90°, 45°, and finally to 0°.
