##### PixoBit Readme #####

1. Press button A, DUT screen will show "Button A pressed" and play beep
2. Press button B, DUT screen will show "Button B pressed" and play beep
3. Cover the light sensor, light value will go down
4. Use flash light point directly to light sensor, light value will go up
5, Touch on pin 0,1, or 2, DUT screen will show "Touched on pin 0" 1, or 2.
