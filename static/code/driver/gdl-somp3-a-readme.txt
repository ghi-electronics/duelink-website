##### MP3 #####

1. Make sure the status LED on the DUT is blinking.
2. Connect a speaker or headphones to the device using a 3.5mm audio jack.
3. Ensure music is playing, using an SD card prepared as follows:

(*) Prepare the SD card with a music file:

- The SD card must be formatted as FAT32, with a maximum size of 32GB.
- Create a folder named 01 in the root directory of the SD card.
- Rename your music MP3 file to 001.mp3.
- Copy 001.mp3 into the 01 folder.
