##### TFT CP23 readme #####

1. Make sure the status LED is blinking.
2. Make sure the text "This is red" is displayed in red 
3. Make sure the text "This is green" is displayed in green 
4. Make sure the text "This is blue" is displayed in blue 
5. Make sure the text "This is white" is displayed in white
6. For touch version only: Touch the (+) at each of the four corners. The (+) disappears when the touch is detected in the correct position.
