##### Chrono #####

Insert the fully charged battery into the battery holder before starting the test.

1. Make sure all LEDs are blinking (25 LEDs).
2. Press Button A. The log box and screen on the test kit will display: “Btn A pressed!”
3. Press Button B. The log box and screen on the test kit will display: “Btn B pressed!”
4. Disconnect the cable from the U socket. Make sure all LEDs are still blinking
