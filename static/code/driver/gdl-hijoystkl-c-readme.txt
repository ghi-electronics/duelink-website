##### Fingerprint readme #####

1. Make sure the status LED is blinking.
2. Move the joystick and make sure the values change in a way that makes sense.
2. Move the joystick all the way to the left, down, up, and right (the order is not important; what matters is that up, down, left, and right are each reached at least once).
3. Press the button.
4. A text message, “Test passed!” is displayed once up, down, left, and right have been detected and the button has been pressed.
