##### LED S284 Readme #####

1.Make sure the status LED is blinking.
2.The LED displays 0000, 1111, 2222, ..., up to 9999. This ensures that each digit column can show values from 0 to 9.
3.For the dot version, make sure all four dots can be turned on and off.
4.For the colon version, make sure the colon can be turned on and off.
5.Make sure the brightness looks good.
