##### RFID #####

1. Ensure the status LED is blinking.
2. Hold the card close to the RFID panel.
3. Confirm that the card is detected and read successfully. The log will display the card number and show the message: "Card read successfully."
4. The test kit screen will also display the reading status and show the number of detection attempts.

(*)The test kit screen does not display the card number due to limited screen space.
