##### SmartLED #####

1. Make sure the status LED is blinking.
2. Inject external 5V and GND to module through Special wires kit
3. Make sure these LEDs are blinking.

Note: Before testing, ensure the following proper connections:
- Requires an LPD8806 LED strip with a minimum of 4 LEDs.
- A special wire kit is needed to connect the LED strip to the module.
- Inject external 5V and GND into the module through the special wire kit.
