##### CincoBit Driver #####

1. Make sure status led is blinking
2. Press U button, Beep will play and Bulb light is red
3. Press D button, Beep will play and Bulb light is green
4. Press L button, Beep will play and Bulb light is blue
5. Press R button, Beep will play and Bulb light is white
6. Check on the screen of DUT: Make sure temperature is same as room temp (25~27)
7. Move DUT, x,y,z value on DUT will be changed
8. Cover light sensor, light value decreases, down to 0
9. Use flash light points directly to light sensor, light value increases, up to 100
