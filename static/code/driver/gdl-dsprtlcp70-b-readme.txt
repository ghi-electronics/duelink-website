##### Portal CP70 readme #####
 
1. Make sure the status LED is blinking.
2. Make sure the screen displays the pattern of colors: White, Yellow, Cyan, Green, Magenta, Red, Blue
3. Make sure the text DUELink appears in the center of the screen.
4. Make sure everything displayed on the screen is clear, with no visual noise.
5. A beep will be played three times after the picture appears on the screen.
6. For Touch version only: Touch on screen, log output text box show X,Y values
