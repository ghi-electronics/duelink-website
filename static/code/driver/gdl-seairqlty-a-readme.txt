##### AirQuality #####

1. Make sure the status LED is blinking.
2. The test screen kit displays the current temperature (Celsius) and humidity. 
3. Ensure the default temperature and humidity values make sense (i.e., they match the room's temperature and humidity).
4. When changing the temperature and humidity of the environment, the displayed values must update accordingly and remain reasonable.
5. The screen updates to the latest values every 2 seconds.
