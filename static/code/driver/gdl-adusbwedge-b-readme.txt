##### USB WEDGE #####

1. Make sure PC has speaker connected, and volume is on
2. Conneck D socket of USB Wedge to the wire has NO knot from the test kit.
3. Connect USB Wedge to PC (by usb port directly, or using USB cable extend), make sure PC plays connect sound.
4. After connected about 1 second, the Programmer will show two pictures: USB Wedge on left side, OLED 096 on right side.
5. Disconect usb cable, make sure PC plays disconnect sound, and the picture on right side will show "No device connected"
