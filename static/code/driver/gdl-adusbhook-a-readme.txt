##### USB HOOK #####

1. Make sure PC has speaker connected, and volume is on
2. Conneck D socket of USB Hook to the wire has NO knot from the test kit.
3. Connect USB Hook to PC by using USB cable, make sure PC plays connect sound.
4. After connected about 1 second, the Programmer will show two pictures: USB Hook on left side, OLED 096 on right side.
5. Disconect usb cable, make sure PC plays disconnect sound, and the picture on right side will show "No device connected"
