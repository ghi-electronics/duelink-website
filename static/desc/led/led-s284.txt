The LED S284 is a 4-digit 7-segment display module that provides bright, clear numeric display capabilities. This module features individual LED segments arranged in a traditional 7-segment configuration across four digit positions.

## Features

- **4-Digit Display**: Show numbers from 0000 to 9999
- **Bright LED Segments**: High-visibility red LEDs for excellent readability
- **Individual Segment Control**: Full control over each segment for custom patterns
- **Decimal Point Support**: Each digit includes a controllable decimal point
- **Easy Integration**: Simple DUELink interface for immediate use

## Applications

Perfect for:
- Digital clocks and timers
- Counter displays
- Temperature readouts
- Score displays
- Industrial monitoring equipment

## Technical Specifications

- **Display Type**: Common cathode 7-segment LED
- **Digits**: 4
- **Color**: Bright red
- **Voltage**: 3.3V/5V compatible
- **Current**: Low power consumption
- **Interface**: DUELink compatible

The LED S284 provides an ideal solution for any project requiring clear numeric display with the convenience of DUELink's simple programming interface.
